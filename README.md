# 🎮 Tic Tac Toe

A modern Tic Tac Toe game built using React, Vite, and Tailwind CSS.

## ✨ Features

- Two Player Game
- Winner Detection
- Draw Detection
- Current Player Indicator
- Reset Game
- Score Board
- Reset Score Board
- Responsive UI

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## 🚀 Run Locally

```bash
npm install
npm run dev
```
